# Soul Sense EQ Test & Journal

The Soul Sense EQ Test is a desktop-based application built using Python's **Tkinter GUI** and **SQLite**. It offers an interactive Emotional Intelligence (EQ) assessment with 20 self-reflective questions and provides a total EQ score with an interpretation. Additionally, it includes a **Mood Journal** feature for daily emotional reflections with AI-powered sentiment analysis and emotional pattern tracking.
